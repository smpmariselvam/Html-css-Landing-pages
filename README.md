## ✨ Features

- **Fully Responsive** - Optimized for desktop, tablet, and mobile devices
- **Modern UI/UX** - Clean and elegant design with smooth animations
- **CSS Grid & Flexbox** - Modern layout techniques for optimal performance
- **Hover Effects** - Interactive card animations and button transitions
- **Pure HTML/CSS** - No frameworks or libraries required
- **Google Fonts** - Playfair Display & Poppins typography
- **SVG Icons** - Scalable vector icons throughout
